```markdown

- [ ] Check today's tasks
- [ ] Review previous day's work

- [ ] Test as you go
- [ ] Commit frequently
- [ ] Log progress in notes

- [ ] Mark tasks complete
- [ ] Commit final changes
- [ ] Note any blockers
```
