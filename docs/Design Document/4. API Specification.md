# API Specification

**Version:** 1.1  
**Base URL:** `https://api.betting-coordinator.com/v1`  
**Authentication:** Bearer JWT Token  
**Last Updated:** January 27, 2026

## Authentication

### Obtain Token
```http
POST /auth/login
Content-Type: application/json

{
  "email": "admin@example.com",
  "password": "secure_password",
  "device_id": "optional_device_uuid"
}
```

**Response:**
```json
{
  "token": "eyJ0eXAiOiJKV1QiLCJhbGc...",
  "expires_at": 1706486400,
  "user": {
    "id": "uuid",
    "email": "admin@example.com",
    "role": "admin"
  }
}
```

### Refresh Token
```http
POST /auth/refresh
Authorization: Bearer {token}
```

## Teams API

### List Teams
```http
GET /teams?status=active&role=PRIMARY
Authorization: Bearer {token}
```

**Response:**
```json
{
  "data": [
    {
      "id": "uuid",
      "name": "Team Alpha",

```

### Create Team
```http
POST /teams
Authorization: Bearer {token}
Content-Type: application/json

{
```

## WebSocket Events

### Subscribe to Round Events
```javascript
// Client connects to WebSocket
const ws = new WebSocket('wss://api.betting-coordinator.com/ws');

// Authenticate
ws.send(JSON.stringify({
  type: 'auth',
  token: 'jwt_token'
}));

// Subscribe to device-specific channel
ws.send(JSON.stringify({
  type: 'subscribe',
  channel: 'private-device.{device_id}'
}));
```

#### Round Prepared

```json
{
```

## Error Responses

### Standard Error Format
```json
{
  "error": {
    "code": "INSUFFICIENT_CAPITAL",
    "message": "Account balance too low for allocation",
    "details": {
      "account_id": "uuid",
      "required": 100.00,
      "available": 50.00
    }
  }
}
```

### Common Error Codes
- `UNAUTHORIZED` - Invalid or expired token
- `VALIDATION_ERROR` - Request validation failed
- `INSUFFICIENT_CAPITAL` - Not enough funds
- `IP_CONFLICT` - IP usage rule violation
- `ROUND_ALREADY_EXECUTING` - Cannot modify executing round
- `DEVICE_OFFLINE` - Device not connected
- `TIME_SYNC_REQUIRED` - Device time out of sync

## Rate Limiting

- **API Endpoints**: 1000 requests/minute per token
- **WebSocket Messages**: 100 messages/minute per connection
- **Time Sync**: 10 requests/minute per device

**Rate Limit Headers:**
```http
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 950
X-RateLimit-Reset: 1706400060
```
